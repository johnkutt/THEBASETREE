# TheBaseTree Architecture

## System Overview

TheBaseTree is a multi-layer green credit ecosystem on Base Chain.

## Layers

1. **Smart Contracts**: Core marketplace logic
2. **Inception Framework**: Dream-layer memory system
3. **Ecosystem Framework**: Regional and circular economy
4. **Frontend**: React + Wagmi interface

## Data Flow

```
User → Frontend → Smart Contracts → Blockchain
                    ↓
            Inception Layer
                    ↓
            Ecosystem Framework
```

## Key Design Decisions

- Base Chain for low gas costs
- Flashblocks for 200ms settlement
- Native AA for agent automation
- Modular architecture for extensibility
